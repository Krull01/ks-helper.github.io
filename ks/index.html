Інформація щодо послуг та вартості

Скільки це коштує

Вартість послуги — 200 грн, незалежно від обсягу робіт. Щоб замовити послугу, пересвідчіться, що на вашому особовому рахунку достатньо коштів, адже плата за Виклик майстра зніметься саме з нього.

Будь ласка, зверніть увагу. Деякі додаткові послуги оплачуються окремо.

Найчастіше запитують

Що входить до вартості послуги?

  • Візит та консультація з питань Домашнього Інтернету і телебачення
  • Діагностика обладнання за адресою підключення
  • Базове налаштування пристроїв: Wi-Fi роутер (без перевстановлення програмного забезпечення), PON injector, ONU-термінал
  • Демонстрація працездатності послуги Домашній Інтернет та гігабітної швидкості (наприклад, вимір швидкості; демонстрація працездатності телебачення/інтернету, надання консультації щодо пошуку телеканалів та роботи ТВ-тюнера для телебачення)
  • Налаштування Київстар ТБ на ПК/ноутбуці, мобільних пристроях з ОС Android/iOS, SMART-телевізорах Samsung і LG, а також ТВ-тюнерах і SMART-телевізорах з Android TV

Инструкции по тестированию скорости

Terminal - iperf3

# меряем upload на 100Мб

iperf3 -c 193.41.63.202 -t 10 -u -b 100M


# меряем download на 100Мб

iperf3 -c 193.41.63.202 -t 10 -u -b 100M -R


# меряем upload на 1Гб

iperf3 -c 193.41.63.202 -t 10 -u -b 1G


# меряем download на 1Гб

iperf3 -c 193.41.63.202 -t 10 -u -b 1G -R

Установка Iperf3

🪟Windows

  • Скачайте архив с официального сайта.
  • Распакуйте файлы (iperf3.exe и cygwin1.dll) в любую папку, например C:\iperf3\.
  • Откройте командную строку (Win+R -> cmd), перейдите в эту папку: cd C:\iperf3 и запускайте команды.

🍎MacOS

  • Самый простой способ — использовать Homebrew. Откройте Терминал.
  • Если Homebrew установлен, просто введите:
    • brew install iperf3
  • После этого iperf3 готов к работе из любого места в терминале.

🐧Linux

  • Программа есть в большинстве стандартных репозиториев.
  • Для Ubuntu / Debian введите в терминале:
    • sudo apt update && sudo apt install iperf3
  • Для CentOS / RHEL: sudo yum install iperf3

📱Android

  • Скачайте приложение Iperf3.
  • ⬇️ Google Play
  • Откройте приложение, перейдите в Advanced Test, введите адрес сервера и запустите тест.
